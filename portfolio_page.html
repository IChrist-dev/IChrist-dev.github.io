E-Commerce Web Store

2023

An ASP.NET web application where customers can browse a product catalogue, adding items to their shopping cart.

This web app simulates a full checkout procedure, validating form content before sending orders to an API for processing. Customers can browse programmatically generated product detail pages and add or remove items from their cart.

Products are stored in a database and managed via an admin panel allowing for full CRUD functionality. All interactions are managed via the Entity Framework.
